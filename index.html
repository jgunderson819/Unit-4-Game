Crystal Collector

You are given a random number as a target number at the start of the game.

There are four crystals below. By clicking on a crystal, you will add a specified number of points to your total score. You win the game, by matching your total score to the target number. You lose if your total score goes above the target number.

Each time a new game starts, the values of each crystal changes but remains the same during each game. Let's see how you can do.
